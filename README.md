ChainTheory-AI


ChainTheory-AI is a rule-based educational reasoning engine 
that identifies common mathematical misconceptions by matching student work against a library of pedagogical rules. 
It is designed to assist teachers in diagnosing errors, not to replace teacher judgment or generate answers.

**Input**

Expression: `10 - 2 × 4`  
Student Answer: `32`  
Student Work: “I did 10 minus 2 first, then multiplied.”
